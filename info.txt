
Criar o projeto: dotnet new web -n <nome do projeto>

Startar projto: dotnet run ou dotnet watch run

Entity frameWork: nuget.org, Microsoft.EntityFrameworkCore,
Microsoft.EntityFrameworkCore.Design, Microsoft.EntityFrameworkCore.Tools,
Pomelo.EntityFrameworkCore.MySql.

apos configuração do contexto e string de conexão

Criar migrations: dotnet ef migrations add AdministradorMigration
persistir no banco de dados: dotnet ef database update

criando seed: dotnet ef migrations add SeedAdministrador
persistir no banco de dados: dotnet ef database update
